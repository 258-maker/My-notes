# 什么是从模式控制器
![](../图片/IMG_20260220_104644.jpg)
- 既可以通过TRGI输入控制信号控制时基单元，又可以通过时基单元的状态来发，经过TRGO发送控制信号

# 8种从模式
| 模式编号 | 模式名称    | 功能描述                      |
| ---- | ------- | ------------------------- |
| 1    | 从模式禁止   | 不通过TRGI接收控制信号             |
| 2    | 编码器模式1  | 与编码器有关                    |
| 3    | 编码器模式2  | 与编码器有关                    |
| 4    | 编码器模式3  | 与编码器有关                    |
| 5    | 复位模式    | TRGI上升沿对CNT清零             |
| 6    | 门模式     | TRGI=高，总开关闭合；TRGI=低，总开关断开 |
| 7    | 触发模式    | TRGI上升沿闭合总开关              |
| 8    | 外部时钟模式1 | TRGI作为时基单元的时钟来源           |

# 8种主模式
| TRGO 模式编号 | 模式名称        | 功能描述                       |
| --------- | ----------- | -------------------------- |
| 1         | 复位          | 手动产生 Update 事件时 TRGO 输出脉冲  |
| ==2==     | ==使能==      | ==总开关闭合，TRGO=1；反之 TRGO=0== |
| ==3==     | ==Update==  | ==每个 Update 事件，TRGO 输出脉冲== |
| 4         | 比较脉冲        | 与输出比较通道 1 有关               |
| 5         | 比较 — OC1REF | OC1REF 作为 TRGO             |
| 6         | 比较 — OC2REF | OC2REF 作为 TRGO             |
| 7         | 比较 — OC3REF | OC3REF 作为 TRGO             |
| 8         | 比较 — OC4REF | OC4REF 作为 TRGO             |
