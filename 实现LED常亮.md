# LED
- 正极引脚较长，内电极较小；负极引脚较短，内电极较大；
- 排阻上的102=1K，即10X10^2，473=47X10^3;

# 点亮LED
- 通过开发板原理图可以知道控制LED灯亮灭的是P2口的，并且接入+VCC5V即高电平1，要LED灯亮的话，P2端要设置为0，不亮就设置为1，有电势差，即有电流流过灯亮；
