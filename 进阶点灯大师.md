# LED闪烁
- 用STC可以生成延迟函数，复制生成代码到主函数上面。其中_nop_()，表示空语句，什么都不做，这个函数要加头文件#include <INTRINS.H>;
```c
#include <REGX52.H>
#include <INTRINS.H>
void Delay500ms(void)	//@11.0592MHz
{
	unsigned char data i, j, k;

	_nop_();
	i = 4;
	j = 129;
	k = 119;
	do
	{
		do
		{
			while (--k);
		} while (--j);
	} while (--i);
}

void main()
{
	while(1)
	{
		P2=0xFE;
		Delay500ms();
		P2=0xFF;
		Delay500ms();
	}
}
```
# LED流水灯
- 生成延迟1毫秒函数，复制到main()函数前面，修改生成的延迟函数，可设置控制延时时间，代码如下
```c
void Delay1ms(unsigned int xms)	//@11.0592MHz
{
	unsigned char data i, j;
	while(xms)
	{
		i = 2;
		j = 199;
		do
		{
			while (--j);
		} while (--i);
		xms--;
	}
}
```
