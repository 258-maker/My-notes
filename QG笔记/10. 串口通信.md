# 通信的基本概念
- 串行：数据按**位（bit）** 为单位，沿**一根数据线** 依次传输（1 位接 1 位）
- 并行：数据按**字节（Byte）** 为单位，沿**多根数据线** 同时传输（8 位 / 16 位一起传）
- 全双工：通信双方可以在同一时刻互相传输数据
- 半双工：通信双方可以互相传输数据，但必须分时复用一根数据线
- 单工：通信只能有一方发送到另一方，不能反向传输
- 异步：通信双方各自约定通信速率
- 同步：通信双方靠一根时钟线来约定通信速率
- 总线：连接各个设备的数据传输线路（类似于一条马路，把路边各住户连接起来，使住户可以相互交流）
## 通信速率（比特率）
- 比特率单位：位/秒（bps）
- 假设一秒发送240个字符，假设一个字符由一个起始位、八个数据位、一个停止位组成，那么比特率为（240X10）/1，即2400bps
## 波特率
- 每秒钟传输了多少个码源
- 基本可以把波特率看成比特率

# 串口通信简介
## 接口标准
- 接口：TXD、RXD、GND；
- 接法：TXD——RXD、RXD——TXD、GND——GND；
- ==通信协议：通常遵循96-N-8-1格式即（9600bps，N——无校验位（奇校验、偶校验、无校验），8——8位数据位，1——停止位长度）==

## 串口内部结构
- TXD（P3.1），RXD（P3.0）
- SBUF：数据缓存区
- 波特率产生是由TH1和TL1（定时器1的功能）产生，配置方式2
- SMOD标志位控制是否使用倍频，置1倍频。

# 串口相关寄存器
## 串口控制寄存器SCON
- SM0、SM1：用来确定串口通信的工作方式，通常使用方式1
- SM2：多机通信控制位，主要是方式2，方式3，为1时来控制RB8为0、为1以此来达到多机通信的目的
- RB8：为1是表示激活RI，为0时表示不激活RI
- TB8：为1是表示激活TI，为0时表示不激活TI
- REN：允许接收位
- TI：发送中断请求标志位，在方式0，当串行发送数据第8位结束时，由内部硬件自动置位，即T1=1时，向主机请求中断，响应中断后必须用软件复位，即TI=0。在其他方式中，则在停止位开始发送时由内部硬件置位，必须用软件复位。初始化配置给0。
- RI：接收中断请求标志位，在方式0，当串行接收到第8位结束时，由内部硬件自动置位，即RI=1，向主机请求中断，响应中断后必须用软件复位，即RI=0。在其他方式中，串行接收到停止位的中间时刻由内部硬件置位，即RI=1（例外情况见SM2说明），必须由软件复位，即RI=0。初始化配置给0。
**设置为：01010011**
## 电源控制寄存器PCON
- SMOD：对倍频的选择
# 计算串口波特率
方式0的波特率= fosc / 12
方式2的波特率=（2SMOD/64）X fosc
方式1的波特率=（2SMOD/32）X （T1溢出率）
方式3的波特率=（2SMOD/32）X （T1溢出率）
**其中T1溢出率=fosc / 12[256-(TH1)]** TH1为初值

# 串口配置步骤
1. 确定串口工作方式（SCON寄存器）；
2. 确定T1的工作方式（TMOD寄存器）；
3. 计算T1的初值（设定波特率），装载TH1、TL1；
4. 启动T1（TCON中的TR1位）；
5. 如果使用中断，需开启串口中断控制位（IE寄存器）

```c
void uart_init(u8 baud)
{
	SCON=0x50;//设置为工作方式1
	PCON=0x80;//波特率加倍
	TMOD|=0x20;//设置计数器工作方式2
	TH1=baud;//计数器初始值设置
	TL1=baud;
	ES=1;//打开接收中断
	EA=1;//打开总中断
	TR1=1；//打开计数器
}
```

# 代码实战
1. 单片机循环向电脑发送 “Hello 51!”
```c
#include <REGX52.H>
#include "delay.h"
#include "uart.h"
void main()
{
	UART_Init();
	while(1)
	{
		UART_SendString("Hello 51!\r\n");
		Delay(100);
	}
}

#include <REGX52.H>
void Delay(unsigned int xms)	//@11.0592MHz
{
	unsigned char data i, j;
	while(xms)
	{
		i = 11;
		j = 190;
		do
		{
			while (--j);
		} while (--i);
		xms--;
	}
}

#include <REGX52.H>
void UART_Init()
{
	SCON=0x40;
	PCON|=0x80;
	TMOD&=0x0F;
	TMOD|=0x20;
	TL1 = 0xF4;
	TH1 = 0xF4;
	ET1 = 0;			//禁止定时器中断
	TR1 = 1;
}

void UART_SendByte(unsigned char Byte)
{
	SBUF=Byte;
	while(TI==0);
	TI=0;
}

void UART_SendString(unsigned char *str)
{
	while(*str!='\0')
	{
		UART_SendByte(*str);
		str++;
	}

}
```
2. 电脑发送指令控制单片机 LED 亮灭。
```c
//之前不用的是时钟中断，这里使用的中断是串口中断
#include <REGX52.H>
#include "Delay.h"
#include "UART.h"
/**
  * @brief 电脑发送指令控制单片机 LED 亮灭,发送文本模式下的“1”P2_0亮，发送“0”P2_0灭
  * @param 
  * @retval 无
  */
void main()
{
	UART_Init();
	
	while(1)
	{
		
	}

}

void UART_Routine(void)  interrupt 4
{
	unsigned char i;
	if(RI==1)//RI是接收控制器，这里用if是因为发送也会触发中断
	{
		i=SBUF;
		if(i=='1')
		{
			P2_0=0;
		}
		if(i=='0')
		{
			P2_0=1;
		}
		RI=0;
	}
}

#include <REGX52.H>
/**
  * @brief   串口初始化
  * @param   无
  * @retval  无
  */
void UART_Init()
{
	SCON=0x50;//REN=1允许中断
	PCON |= 0x80;
	TMOD &= 0x0F;			
	TMOD |= 0x20;			
	TL1 = 0xF4;		//设置定时初值		
	TH1 = 0xF4;		//设置定时器重装值								
	ET1=0;   //禁止定时器1中断
	TR1 = 1; //定时器1开始计时
	EA=1;//启动中断
	ES=1;//启动串口中断
}
/**
  * @brief 串口发送一个字节数据  
  * @param  Byte 要发送的一个字节数据
  * @retval 无
  */
void UART_SendByte(unsigned char Byte)
{
	SBUF=Byte;
	while(TI==0);
	TI=0;
}
```
